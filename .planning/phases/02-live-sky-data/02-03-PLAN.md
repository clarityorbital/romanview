    Start the dev server with `npm run dev` and open the app in browser. Run through all nine verification tests below. This is a human verification checkpoint -- no code changes needed.

    **what-built:** Complete Phase 2: Live Sky Data integration -- target name resolution, HMS/DMS coordinate input, dual-format coordinate display, live Gaia DR3 star queries with adaptive density management, star rendering in both 3D sky view and focal plane SVG, and density indicator.

    **how-to-verify:**

    **Test 1: SIMBAD name resolution (TARG-01)**
    1. In the target search box (SIMBAD mode), type "M31"
    2. Wait for results to appear (~1-2 seconds)
    3. Click the result to add the target
    4. EXPECTED: View centers on M31 (RA ~10.68, Dec ~+41.27), coordinates show in the display

    **Test 2: HMS/DMS manual input (TARG-02)**
    1. Click [RA/DEC] to switch to manual mode
    2. Enter "00h42m44.3s +41d16m09s" in the coordinate field
    3. Click ADD TARGET
    4. EXPECTED: Target is added at approximately RA=10.685, Dec=+41.269

    **Test 3: Decimal degree input (TARG-02 continued)**
    1. Still in manual mode, enter "266.417 -29.008" (near galactic center)
    2. Click ADD TARGET
    3. EXPECTED: Target is added at those coordinates

    **Test 4: Dual coordinate display (TARG-03)**
    1. Select any target
    2. Look at the coordinate display panel (bottom-left)
    3. EXPECTED: TARGET section shows BOTH:
       - HMS/DMS format (e.g., 00h 42m 44.3s / +41d 16' 09")
       - Decimal degrees (e.g., 10.6847 / +41.2687)

    **Test 5: Gaia stars in sky view (STAR-01)**
    1. Select M31 target
    2. Wait a few seconds for Gaia query
    3. Look at the 3D sky view near the target
    4. EXPECTED: Blue Gaia star points appear near the target position, separate from the white/warm HYG background stars

    **Test 6: Gaia stars in focal plane (STAR-01 continued)**
    1. With M31 selected, look at the focal plane view (top-left panel)
    2. EXPECTED: Blue star dots appear at positions across the detectors

    **Test 7: Dense field adaptive limiting (STAR-02)**
    1. Search for "Sgr A*" or add galactic center coordinates (266.417, -29.008)
    2. Wait for Gaia query
    3. EXPECTED: Focal plane footer shows ~2000 sources with a magnitude limit significantly brighter than 18 (e.g., "2000 sources (G < 14.5)"). May show [DENSE] indicator.

    **Test 8: Sparse field (STAR-02 continued)**
    1. Search for "NGC 1277" or add a high-latitude sparse field (e.g., 49.95, +41.58)
    2. Wait for Gaia query
    3. EXPECTED: Fewer stars (maybe 50-500), with a deeper magnitude limit (e.g., "87 sources (G < 21.0)")

    **Test 9: Density indicator (STAR-03)**
    1. With any target selected, look at the focal plane footer
    2. EXPECTED: Shows "{N} sources (G < {M.m})" with the actual source count and effective magnitude limit
    3. During query, should briefly show "Querying..."
